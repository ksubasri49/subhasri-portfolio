# Subha Sri K — Portfolio

A professional personal portfolio website built from résumé content.

## Folder Structure

```
portfolio/
├── index.html        ← Main portfolio page (open this in a browser)
├── style.css         ← All styles
├── script.js         ← Navigation, animations & interactions
└── assets/
    ├── photo.jpeg    ← Profile photo
    └── resume.pdf    ← Downloadable résumé
```

## How to Use

1. **Locally**: Open `index.html` directly in any modern browser.
2. **Free hosting**:
   - **Netlify**: Drag and drop this folder at netlify.com/drop — get a live link instantly.
   - **GitHub Pages**: Push to a GitHub repo → Settings → Pages → Deploy from main branch.
   - **Vercel**: Connect your GitHub repo or drag-drop the folder.

## What's Included

- Hero section with photo, intro, and quick stats (CGPA, projects, semester)
- Skills section: languages (with proficiency bars), frameworks, tools, soft skills, coursework
- Three projects: SOULNEST (hackathon, featured), Django E-Commerce App, Resume Relevance Checker
- Experience & certifications timeline, including the Java course and hackathon
- Education banner with circular CGPA indicator
- Contact section with email, phone, LinkedIn, GitHub, and résumé download
- Fully responsive, mobile menu included

## Customization

- Edit `index.html` for content changes.
- Edit `style.css` — all colors are CSS variables at the top of the file (`:root`).
- Replace `assets/photo.jpeg` or `assets/resume.pdf` with updated files (keep the same filenames), or update the `src`/`href` in `index.html` if you rename them.
